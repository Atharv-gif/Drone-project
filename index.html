Aerial Drone through ESP32

This project focuses on creating an aerial drone controlled through ESP32, a low-cost, low-power microcontroller with integrated Wi-Fi and Bluetooth capabilities. The drone will be capable of performing various functions such as aerial photography, live data streaming, and automated flight using pre-programmed routes.

Project Goals

  • Develop a stable and efficient control system using ESP32.
  • Integrate sensors for obstacle detection and navigation.
  • Enable remote control via mobile app or computer.

Technologies Used

  • ESP32 for control and communication.
  • Drone hardware (motors, propellers, frame).
  • Wi-Fi and Bluetooth communication protocols.
